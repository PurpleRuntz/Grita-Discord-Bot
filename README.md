## 📋 Overview

**Grita** is an all-in-one Discord bot designed to transform your server into an interactive gaming and community hub. With features ranging from server moderation to economy systems, team battles, music streaming, and entertainment, Grita brings endless possibilities to your Discord experience.

## ✨ Key Features

- 🛡️ **Advanced Moderation** - Complete server management tools
- 💰 **Economy System** - Virtual currency, banking, and trading
- 🎮 **Mini-Games** - Blackjack, slots, roulette, and more
- 👥 **Team System** - Create teams, battle, and compete
- 🎵 **Music Player** - Play music from various sources
- 🖼️ **Image Filters** - Apply effects and generate memes
- 📊 **Invite Tracking** - Monitor server growth
- 🏆 **Ranking System** - Reputation and leveling

## 🚀 Quick Start

1. **[Invite Grita to your server](YOUR_INVITE_LINK_HERE)**
2. Ensure Grita has the necessary permissions
3. Type `!help` to see all available commands
4. Start using commands like `!daily` to begin your economy journey!

## 🎯 Getting Started with Economy

1. **Claim Daily Bonus**: `!daily` - Get coins every 12 hours
2. **Check Balance**: `!bal` - See your coins and bank balance
3. **Work for Money**: `!work` - Earn coins with 1-hour cooldown
4. **Deposit Safely**: `!deposit all` - Keep your coins safe from thieves
5. **Buy Jobs**: `!job` - Purchase jobs with reputation points
6. **Start Gaming**: Try `!slots 100` or `!blackjack 50`
